#README

To run the program:
 1. type "make" to compile the program
 2. on the host hostX: open one window and type "./nse-linux386 hostX port"
 3. on the host hostY: open 5 windows for 5 routers, and on each window
 	- type "java router 1 hostX port port1"
	- type "java router 2 hostX port port2"
	- type "java router 3 hostX port port3"
	- type "java router 4 hostX port port4"
	- type "java router 5 hostX port port5"
